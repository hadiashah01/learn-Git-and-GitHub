# Repository Creation using Git (Local Method)
This document explains the complete process of creating and managing
a Git repository locally and pushing it to GitHub using Git commands.
All steps are supported with screenshots.

---

## 1. Git Initialization
**Command:**

![Git Init Screenshot](git%20init.jpeg)

**Description:**

Initializes an empty Git repository in the current directory.
## 2. Git Status

**Command:**

![Git Status Screenshot](git%20status.jpeg)


**Description:**

Displays the current state of the working directory and staging area.

## 3. Add Files to Staging Area

**Command:**

![Git Status Screenshot](git%20add.jpg)


**Description:**

Stages all modified and new files for commit.

## 4. Commit Changes

**Command:**

![Git Status Screenshot](git%20commit.jpeg)


**Description:**

Creates a snapshot of staged changes.

## 5. Push Code to GitHub

**Command:**

![Git Status Screenshot](git%20push.jpeg)


Description:

Uploads local commits to the remote repository.

![Git Status Screenshot](git%20pull.jpeg)

**Command:**

![Git Status Screenshot](git%20clone.jpeg)
